# SkillOps Local v0.1.2

> 发布前请先确认 `package.json` 与 `wails/main.go` 中版本号已同步到 `0.1.2`。

## 中文发布说明（GitHub/Release）
- 增加「桌面工作台」界面，拆分为 5 个可独立页面：
  - 仪表盘
  - 扫描与风险
  - MCP 管理
  - 更新中心
  - 签名与安装器
- 加强 Wails 原生能力：
  - 增加菜单项（打开页面、检查更新、重启服务、签名查询、安装器目录）
  - 服务状态、安装目录、版本、签名状态联动展示
  - 提供本地服务重启和打开本地子页能力
- 新增桌面更新检测能力：读取 GitHub Releases 最新版本并对比当前版本
- 新增签名与安装器入口：
  - 获取当前签名信息
  - 安装器输出目录与 `.pkg/.dmg` 产物说明
  - 支持打开安装器目录
- 文档与发布支持：
  - 新增 App Store 商品元信息模板
  - 补充发布说明、截图命名与简介文案（中英）

说明：本次变更仅包含用户可见产品能力和发布文案，不包含 AI 提示词开发内容。

## What's New (GitHub/Release)
- Added a multi-page desktop workbench with dedicated sections:
  - Dashboard
  - Scan & Risk
  - MCP Management
  - Updates
  - Signing & Installer
- Strengthened Wails native capabilities:
  - New menu actions for page navigation, update checks, service restart, signing status, and installer path
  - Runtime state exposure for local service status, install directories, versions, and signing context
  - Local-page open and service control hooks exposed from desktop UI
- Added desktop update detection via GitHub Releases with current-versus-latest comparison
- Added signing and installer workflow entry points:
  - Signature inspection
  - Installer output paths for `.pkg` / `.dmg`
  - Open installer folder directly from workbench
- Documentation & release readiness updates:
  - Added App Store metadata template
  - Added release copy for App Store/ GitHub in bilingual format

This release avoids internal AI prompt or model-development notes in user-facing content.
